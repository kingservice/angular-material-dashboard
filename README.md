# Angular material admin dashboard

[DEMO](http://micrum.github.io/angular-material-admin/site/)

## Getting started

Clone project:

    $ https://github.com/micrum/angular-material-admin.git

Install dependencies:

    $ npm install
    $ bower install
    
Run development web-server:

    $ gulp serve
